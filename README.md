## 로그인 
DotUs는 본래 카카오톡 프로필 사진을 기반으로 한 서비스이므로, 카카오톡 로그인만을 지원합니다.        
'카카오 로그인' 버튼을 누르면 카카오톡 어플이 설치되어 있을 경우 어플, 아닐 경우 웹 브라우저를 이용해 로그
본 어플에 처음 로그인한다면, 아래 사항에 대한 동의를 받습니다. (* 표시 되어있는 항목은 필수)           
+ 닉네임*
+ 프로필 사진*
+ 카카오계정(이메일)*
+ 카카오톡 메시지 전송
